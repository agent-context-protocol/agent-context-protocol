### Table Formatting Guidelines:

When creating tables, use a clean, professional layout with a white background and contrasting text for readability. Consider these practices:

1. LaTeX Packages:
   ```latex
   \usepackage{booktabs}     % for professional looking lines
   \usepackage{longtable}    % to allow tables to flow across multiple pages
   \usepackage{xcolor}       % to enable row coloring if needed
   \usepackage{array}        % for extended column definitions

	2.	Table Structure:
	•	Use the longtable environment for all tables so that if content is too long, it flows naturally to subsequent pages:

\renewcommand{\arraystretch}{1.3} % Adjust vertical spacing
\begin{longtable}{|p{3cm}|p{3cm}|p{3cm}|}
\caption{\textit{Your Table Caption}} \label{tab:example_table} \\
\hline
\textbf{Header 1} & \textbf{Header 2} & \textbf{Header 3} \\
\hline
\endfirsthead
\caption*{\textit{(Continued from previous page)}}\\
\hline
\textbf{Header 1} & \textbf{Header 2} & \textbf{Header 3} \\
\hline
\endhead

% Table body rows go here
Row 1 Col 1 & Row 1 Col 2 & Row 1 Col 3 \\
\hline
Row 2 Col 1 & Row 2 Col 2 & Row 2 Col 3 \\
\hline

\end{longtable}


	•	Keep column borders where it aids clarity (e.g., vertical lines).
	•	Use booktabs commands such as \toprule, \midrule, and \bottomrule if you prefer a cleaner look with horizontal rules only.

	3.	Row Colors (Optional):
	•	If you want alternating row colors for readability, you can do:

\rowcolors{2}{gray!15}{white}

right after the \begin{longtable}{...} line. Make sure to load the xcolor package with table or svgnames options.

	4.	Highlighting important information:
	•	You can use bold text (\textbf{}) or cell coloring (\cellcolor{yellow}) to emphasize essential data. Use color sparingly.
	5.	Visual groupings within the table:
	•	Use multi-row or multi-column cells when necessary, making sure to keep everything readable. The multirow or multicol packages can help with advanced formatting.
	6.	Keep the background white:
	•	By default, LaTeX tables have a white background. Use minimal color highlights and crisp lines for a professional appearance.
	7.	Captions and References:
	•	Always provide a caption beneath the table using:

\caption{\textit{Description of the table}}


	•	Label the table and reference it in the text as “Table~\ref{tab:example_table}”.

	8.	General Tips:
	•	Adhere to professional and consistent formatting across all tables in the report.
	•	Use color or shading only if it truly enhances readability or highlights crucial points.
	•	Make sure tables fit within page margins. Use vertical elongation (longtable) rather than reducing font size drastically.
  • In table and figures captions, sometimes you are duplicating Table X: Table X....(rest of the caption) or Figure X: Figure X...(rest of the caption), please avoid that.
